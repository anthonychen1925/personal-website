# Personal Website

This is a minimal, responsive personal website scaffold.

Preview locally:

```bash
cd personal-website
python3 -m http.server 8000
# open http://localhost:8000
```

Files:
- index.html — main page
- styles.css — styling
- script.js — small behaviors (theme toggle, simple form)
